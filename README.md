# 前期准备
### 安装(win相关操作)
参考： http://www.runoob.com/mysql/mysql-install.html   
下载sqlserver：https://dev.mysql.com/downloads/mysql/ （最新版本）  
安装版本：mysql-5.5.38-winx64
``` c
cd C:\Program Files\MySQL\MySQL Server 5.5\bin
//启动服务
net start mysql  
//停止服务
net stop mysql
```
